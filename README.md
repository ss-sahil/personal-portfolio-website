# 🎓 BCA Mini Project: Student Portfolio Website

This project is a **Student Portfolio Website** developed using **HTML and CSS** as part of a first-year BCA mini project.

---

## 📌 Project Overview
Created a simple, clean, and responsive portfolio website to showcase personal details, skills, and projects.

---

## ✨ Features
- Clean and structured layout (Header, About, Skills, Projects, Contact)
- Personal introduction with profile details
- Skills section highlighting technical abilities
- Projects section showcasing academic work
- Contact form for basic user interaction
- Beginner-friendly UI with responsive design

---

## 🛠️ Technologies Used
- HTML5
- CSS3

---

## 📚 Learning Outcomes
- Understanding webpage structure and layout design
- Styling using CSS (Flexbox, spacing, colors)
- Creating a personal portfolio website
- Building a responsive multi-section website
